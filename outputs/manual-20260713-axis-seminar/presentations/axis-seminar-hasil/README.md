# AXIS Seminar Hasil 2026

File utama: `AXIS_Seminar_Hasil_2026.pptx`.

Deck berisi 15 slide dengan seluruh teks, kotak, garis, dan diagram ringkas sebagai objek PowerPoint native yang dapat diedit. File dapat dibuka langsung di Microsoft PowerPoint atau diimpor ke Canva melalui **Create a design → Import file**.

## Isi Deck

1. Judul
2. Masalah dan kesenjangan
3. Rumusan masalah
4. Posisi AXIS
5. Kebutuhan rancangan
6. Alur solusi
7. Percakapan reflektif
8. Mood dan PHQ-9
9. Memori dan lifecycle
10. Arsitektur implementasi
11. Pengalaman serta kendali pengguna
12. Strategi evaluasi
13. Hasil validasi fungsional
14. Bukti komparatif
15. Kesimpulan dan evaluasi lanjutan

## Batas Klaim

Hasil fungsional ditampilkan sebagai bukti pengujian komponen. Hasil perbandingan percakapan dan graf ditulis sebagai bukti awal yang bercampur; deck tidak menyatakan AXIS unggul mutlak atas chatbot baseline. Pengujian pengguna nyata dan pengukuran latensi voice ditandai sebagai rencana menuju sidang akhir.
