# Assessment CLI bot (PHQ-9)

Lightweight manual test harness for the PHQ-9 feature.

## Run

From repo root:

```bash
.venv/bin/python agentic/tests/test_feature_bot/test_assessment_bot/cli.py
```

Force a language:

```bash
.venv/bin/python agentic/tests/test_feature_bot/test_assessment_bot/cli.py --language id
```

Start directly in a PHQ-9 phase (useful when debugging a specific state):

```bash
.venv/bin/python agentic/tests/test_feature_bot/test_assessment_bot/cli.py --bootstrap-phase offered --language id
```

## How to use

- Type a normal chat message; PHQ-9 will eventually surface its offer after the warm-up turns.
- When the PHQ-9 options are shown, send a button tap as plain text: `0`, `1`, `2`, or `3`.
- Type `quit` to exit.

## Output

The CLI prints:

- `assistant:`: the message text generated by the PHQ-9 delivery node
- `metadata.phq9:`: a simulated payload matching the shape described in `docs/phq9_frontend_notes.md`
