# Visual References for Seminar Review

Halaman ini adalah potongan visual dari pedoman dan laporan pembanding yang dipakai saat menilai `seminar_hasil.pdf`. Berkas dibuat sebagai gambar terpisah agar struktur halaman, daftar isi, dan pembukaan bab dapat diperiksa tanpa membaca PDF penuh.

Kelompok gambar:
- `00`--`04`: pedoman penulisan.
- `10`--`12`: laporan akhir pembanding (`thesis.pdf`).
- `20`--`21`, `30`--`31`: laporan kakak tingkat.
- `40`--`41`: naskah seminar hasil AXIS.
